# Land & More Reality — Dara Villa Luxury Estate

A premium real estate landing page for **Dara Villa Luxury Estate**, Uyo, Akwa Ibom State, Nigeria.

**Live site:** [https://drdriod.github.io/landmore/](https://drdriod.github.io/landmore/)

## Features

- Hero section with luxury villa imagery
- Plot pricing cards (300sqm · 450sqm · Custom)
- Location details & landmarks
- Land documentation / title security overview
- Inspection booking form (sends via WhatsApp)
- Contact section with bank payment details
- Fully responsive (mobile + desktop)

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Cormorant Garamond & Montserrat fonts

## Development

```bash
pnpm install
pnpm run dev
```

## Deploy to GitHub Pages

```bash
pnpm run build:github
# Commit and push — GitHub Pages serves from /docs
```
